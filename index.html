About

Liron Mor and Roey Bokobza.

We used Jquery to validate the login and sign up forms. the validation is needed for letting the user know if the details he entered are all in form.
In addition, we used it to control and change the settings (what key was clicked).

The template we used in order to locate the divs propelry on the screen and for creating this modal window, is from w3-schools.

The diffculties we had was :
- Validate the login and sign tp forms "on place" and inform the user when he is wrong.
- Locate the game screen and menus propelry on the screen.
- Make the characters look where the move.
- Implemeting the BFS algorithm for all ghosts.

Have fun!
